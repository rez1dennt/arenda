# Task Completion Checklist

Before reporting completion:
- Open the site via local server, usually `http://127.0.0.1:5173/`.
- Check desktop, tablet, and mobile widths for horizontal overflow.
- Check browser console for site-owned errors. Yandex widget may emit external metrika/network errors in test browsers.
- Remove temporary screenshots or generated verification files from the project root.
- Keep user-facing summary short and mention anything that could not be verified.
