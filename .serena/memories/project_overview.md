# Project Overview

Static landing site for quad bike / outdoor recreation rental in Tver Oblast. The project is a plain HTML/CSS/JS site with images in `assets/` and no build system.

Main files:
- `index.html`: page structure and content.
- `style.css`: all layout, responsive styles, and visual design.
- `script.js`: small UI behavior for smooth anchors and button press feedback.
- `assets/`: image assets and map image leftovers.
