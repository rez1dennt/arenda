# Style And Conventions

- Plain HTML, CSS, and vanilla JavaScript.
- CSS uses BEM-like class names such as `hero__content`, `footer__contacts`, `price-table`.
- Visual style: dark rugged landing page, large condensed uppercase typography, yellow/green action accents.
- Responsive behavior is handled with CSS media queries at `1100px`, `760px`, and `560px`.
- Keep edits focused in `index.html`, `style.css`, and `script.js`; avoid adding a framework or build step.
